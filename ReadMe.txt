자동 방치형 게임

기준디스플레이 : FullHD, 갤럭시 Note10(안드로이드)


 - 캐릭터는 자동으로 추적 거리 안의 적을 찾아 추적, 추적 사거리 밖에있을경우 가만히 대기
 - 몬스터는 캐릭터를 발견시 타겟으로 설정후 추적, 타겟과의 거리가 멀어지거나 추적 실패시 정찰모드로 변경
 - 캐릭터는 적을 공격시 일반공격과 스킬을 번갈아가며 사용, 각 공격대기시간과 스킬대기시간에 맞게 실행
예시) 일반공격 -> 스킬대기시간 -> 스킬 -> 일반공격 대기시간 -> 일반공격 -> 스킬대기시간 -> 스킬....
	스킬사용시에는 스킬 대상에게 이펙트효과 발동
 - 캐릭터는 사망시 일정시간 후에 부활, 만약 부활하기도 전에 전멸시 게임오버화면으로 이동
 - 게임오버후 재시작 버튼 누를시 다시 게임화면으로 이동. 처음부터 시작

 - 캐릭터 레벨업 구현 : 캐릭터를 따라다니는 Hp바와 경험치바 추가, 캐릭터 레벨업시 현재의 최대체력만큼 회복
 - 상점구현 : 몬스터 사냥시 골드획득, 상점버튼으로 상점을 열어서 최대체력과 공격력 구입
	단, 최대체력을 구매하여도 바로 회복되지는 않음

 - 스테이지구현 : 일정 몬스터를 사냥시 보스몬스터 생성, 보스 생성시 더이상 몬스터는 나타나지 않음
		보스 사냥 성공시 승리문구가 나타나고 캐릭터들은 승리애니메이션 재생,
		일정시간 지난후 다음스테이지 진입, 진입시 몬스터의 체력,공격력,골드획득량,경험치획득량 증가

 - 안드로이드 빌드 : 갤럭시 Note10기준으로 UI배치, 60프레임고정

 - 캐릭터 정보창 구형 : 화면 왼쪽아래 캐릭터 초상화배치, 클릭시 캐릭터에 대한 최대체력과 공격력 수치로 확인 가능
